# VUEBLOG
## Description
Hello this is my vue3 project.Here you can register, add music, video and share posts with friends
## Setup
Clone the repository
```
git clone https://github.com/AleksandrSmirnov92/VueBlog.git
```
Run these commands
```
cd Backend
npm i
npm run dev
```
